# Tetris – FACULTY Edition

## Como rodar o projeto

1. Crie o ambiente virtual (venv):
   python -m venv venv

2. Ative:
   - Windows: venv\Scripts\activate
   - Linux/Mac: source venv/bin/activate

3. Instale as dependências:
   pip install pygame

4. Execute o jogo:
   python main.py

## Recursos
- Fundo aleatório (coloque imagens em assets/backgrounds/)
- Fonte padrão Tahoma
- Menu interativo e seleção de dificuldade
- Sistema de pontuação simples e intuitivo
